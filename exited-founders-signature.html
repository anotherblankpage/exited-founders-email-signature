How to add this signature in Outlook:
  1. Fill in your details and click Generate Signature.
  2. Click Select Signature to Copy, then press Ctrl+C / Cmd+C.
  3. In Outlook go to File → Options → Mail → Signatures.
  4. Create or edit a signature, click inside the editor, and paste (Ctrl+V / Cmd+V).
  5. Save and set as your default signature.
